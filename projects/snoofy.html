Snoofy Open Source Brand
Anyone can use the Snoofy logo as part of their brand, if they want. Why wouldn't they want? It is released under the CC BY-SA 4.0 license. Note that the license only applies to the logo itself, not the Remachine font by Mans Greback foundry that was used to create it. That would need to be licensed from them.
For example, the Monome Norns project may want to create a Snoofy-branded Norns box.
