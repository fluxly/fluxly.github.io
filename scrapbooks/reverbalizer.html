Reverbalizer

The Modern Device Reverbalizer is a hackable electronic multi-effects module for mobile and microcontroller applications.

It is hackable in two ways. First, you can adapt it for use in a microcontroller project. Second, you can download or create additional DSP effects and install it on an EEPROM cartridge using an Arduino-based program we developed. It comes with 8 effects in ROM and 8 on the re-programmable cartridge.
